# 📰 Django Blog App

Hi 👋, I'm Shivam!

This is a **Blog Web Application** built using Django where users can read, create, and manage blog posts.
I created this project while exploring backend development and understanding how content-driven platforms like blogs actually work behind the scenes.

---

## ✨ Features

* 📝 Create new blog posts
* 📖 Read all posts in a clean layout
* ✏️ Edit existing posts
* ❌ Delete posts
* 🗂️ Organized content display
* 💾 Data stored using SQLite

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite

---

## 📂 Project Structure


env/                # Virtual environment
blog/               # Main Django app
templates/          # HTML templates
db.sqlite3          # Database
manage.py           # Django management script
```

---

## ⚙️ How to Run Locally

Follow these steps:


# Clone the repository
git clone https://github.com/shivampoojanyadav/blog_app.git

# Move into project directory
cd blog-project

# Activate virtual environment
.\env\Scripts\Activate

# Install dependencies
pip install django

# Run migrations
python manage.py migrate

# Start the server
python manage.py runserver
```

Open 👉 http://127.0.0.1:8000
